== Description ==

Give your users to ability to create stickynotes on your blog. Customize colors, size, and number of stickynotes to be displayed at once. Uses wordpress 3's custom post types for easy managing. Is still in infant stages and needs your help to iron out all the bugs

== Installation ==

1. Upload and unzip `wp_sticky_notes.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [wp-sticky-notes] in your templates
